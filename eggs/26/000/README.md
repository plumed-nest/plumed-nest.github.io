**Project description and instructions**  
plumed_trial_start.dat activates an OPES bias for a new simulation. plumed_trial_restart.dat is used for restarting. plumed_prod.dat is used for production simulations with a static bias (e.g., STATE.production). Simulations were performed with GROMACS 2023.4 + PLUMED 2.10.
